## Архитектурные заметки

- Авторизация: JWT (access 15m) + refresh token rotation (7d), httpOnly cookies
- `sendSuccess` / `sendError` — единый формат ответов
- Админ-логи: файлы `logs/combined-YYYY-MM-DD.log`, парсинг через `fetchLogs`
- Сид: admin@test.com / admin; manager@test.com / manager
- Rate limiter: 500 запросов / 15 мин на `/api/`, in-memory
- DB Browser: raw SQL через whitelist таблиц
- ReadOnly поля: password, resetPassword*, currentSessionId, createdAt, updatedAt, lastSeen
- Бэкап: `docker exec projects_postgres_18 pg_dump`, файлы в `server/backups/`
- Статистика онлайн: Socket.IO, `getOnlineUsersFromSockets()`
- Тестовая БД: Docker на порту 5433, `docker compose -f docker-compose.test.yml up -d`
- 1С заглушки: 3 карточки на дашборде партнёра с модалкой "Ожидаем 1С"
